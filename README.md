# Chakra UI Skeleton

## Features

- Chakra v2
- React 18.x
- Typescript
- Composition API
- Theme Manager (Dark Mode)
- Todo Example
- React Hook Form Example
- Authentication with Axios Interceptor

## Installation

```
npm install
```

## Run

```
npm start
```
